Enter Charles Petzold.

Since I had coded last, I'd started the book 'CODE: The Hidden Language of Computer Hardware and Software' by Charles Petzold. It starts with an exercise designed to inspire the reader to count numbers in bases other than 10. Petzold then the general algorithm for calculating the value of a base-10 number into a base-whatever number. It seemed like a pretty enticing thing for me to try and automate myself, so here we go!

Here's the formula to convert any number to base q:

N = xnqn + ... + x1q1 + x0q0

Where

  • N is any integer
  • x is the digit
  • n is the position of the digit (starting with 0 for the "ones" place, 1 for "tens," etc.)
  • q is the base value
